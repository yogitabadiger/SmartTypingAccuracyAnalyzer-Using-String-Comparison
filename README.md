# ⌨️ Smart Typing Accuracy Analyzer Using String Comparison

> An Advanced Java based application for real-time typing performance analysis using string comparison techniques and the Levenshtein Distance Algorithm.


## 🚀 Project Overview

The **Smart Typing Accuracy Analyzer Using String Comparison** is a Java application developed to analyze and improve typing performance. The system compares the original sentence with the typed sentence and provides real-time typing analysis including:

✅ Typing Accuracy
✅ Typing Speed (WPM)
✅ Error Detection
✅ Real-Time Performance Evaluation

The project is built using **Java Swing**, **Advanced Java Concepts**, and **String Comparison Algorithms**.



## ✨ Features

* 🔹 Real-time typing analysis
* 🔹 Typing accuracy calculation
* 🔹 Words Per Minute (WPM) calculation
* 🔹 Error detection using Levenshtein Distance Algorithm
* 🔹 Random sentence generation
* 🔹 Interactive Java Swing GUI
* 🔹 Character-by-character string comparison
* 🔹 User-friendly interface
